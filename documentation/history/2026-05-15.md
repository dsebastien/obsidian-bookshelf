## Fixed CSS lint warnings

- `src/styles.src.css:356` — Replaced `column-gap`/`row-gap` pair on `.bookshelf-books` with the `gap: <row> <col>` shorthand. The reviewer flags `column-gap` because it's part of the partially-supported "multicolumn" CSS feature module (even though it works for flexbox). The `gap` shorthand avoids the warning. Computed layout is identical.
- `src/styles.src.css:514` — Removed `!important` from `.bookshelf-book-property.bookshelf-book-property--rating` color rule. The doubled-class selector already has specificity 0,2,0, which ties `:not(:first-child)` (also 0,2,0) and wins via source order, and beats the base `.bookshelf-book-property` (0,1,0). Added a brief comment explaining the specificity reasoning so a future reader doesn't reintroduce `!important`.
